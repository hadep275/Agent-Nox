‚öôÔ∏è Configuration

Basic Settings

{
  // Primary AI provider
  "nox.aiProvider": "anthropic",
  
  // API Keys (stored securely)
  "nox.anthropic.apiKey": "your-key-here",
  "nox.openai.apiKey": "your-key-here",
  "nox.deepseek.apiKey": "your-key-here",
  
  // Performance settings
  "nox.maxContextSize": 100000,
  "nox.enableCaching": true,
  
  // Monitoring
  "nox.enableTelemetry": true,
  "nox.logLevel": "info"
}

Advanced Configuration

  • maxContextSize: Maximum tokens to send to AI (default: 100,000)
  • enableCaching: Cache responses for faster performance (default: true)
  • enableTelemetry: Anonymous usage analytics (default: true)
  • logLevel: Logging verbosity (debug, info, warn, error)

üîß Troubleshooting

Common Issues

‚ùå "API Key not found" Error

  • Ensure you've set the correct API key in VS Code Settings
  • Check that the key format matches your provider's format
  • Verify the key has sufficient credits/permissions

‚ùå Commands not appearing

  • Restart VS Code after installation
  • Check that the extension is enabled in Extensions panel
  • Look for error messages in the Output panel (View ‚Üí Output ‚Üí Nox)

‚ùå Slow responses

  • Check your internet connection
  • Try reducing nox.maxContextSize
  • Enable caching with nox.enableCaching: true
